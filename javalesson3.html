Java Lesson 3: JAVA VARIABLES

1. int:

This variable type is used for storing whole numbers. It can hold values between -2,147,483,648 and 2,147,483,647.

Example: `int age = 25;`

2. double:

This variable type is used for storing decimal numbers with high precision. It can hold values with a wider range than `int`.

Example: `double pi = 3.14;`

3. boolean:

This variable type is used for storing either true or false values, representing binary conditions.

Example: `boolean isReady = true;`

4. char:

This variable type is used for storing single characters. It is enclosed in single quotes.

Example: `char grade = 'A';`

5. String:

This variable type is used for storing sequences of characters, representing text. It is enclosed in double quotes

Example: `String name = "John";`

6. float:

This variable type is similar to `double` but with lower precision. It is used for storing smaller decimal numbers.

Example: `float weight = 65.5f;`

7. long:

This variable type is used for storing larger whole numbers. It can hold values between -9,223,372,036,854,775,808 and 9,223,372,036,854,775,807.

Example: `long population = 7,896,541,230L;`

Conclusion

Remember that variable names, such as `age`, `pi`, `isReady`, `grade`, `name`, `weight`, and `population`

are chosen by the programmer and can be customized according to the specific context and purpose of the program.
