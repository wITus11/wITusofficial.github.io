HTML Lesson 3: Introduction to CSS Styling

Welcome to the world of CSS (Cascading Style Sheets)!

In this lesson, we will explore the basics of CSS and how it can be used to style and enhance your HTML documents. CSS allows you to control the appearance and layout of your web pages, making them visually appealing and engaging. Let's get started!

1. CSS Syntax:

CSS uses a simple syntax to define styles and apply them to HTML elements. CSS styles consist of a selector and one or more declarations. Selectors target specific HTML elements, while declarations define the properties and values to be applied. Here's an example:

2. CSS Selectors

CSS selectors determine which elements a style should be applied to. There are various types of selectors available, including element selectors, class selectors, ID selectors, and more. Here are some common selector examples:

- Element Selector:

Targets all instances of a specific HTML element

- Class Selector:

Targets elements with a specific class attribute value.

- ID Selector:

Targets a specific element with a unique ID attribute value.
